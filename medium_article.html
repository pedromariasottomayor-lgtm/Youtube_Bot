The Problem With YouTube in 2025

YouTube rewards consistency above almost everything else. Channels that upload daily grow faster, get recommended more, and compound their subscriber count geometrically. But daily content creation is unsustainable for most solo creators.

Think about what goes into one YouTube Short: picking a topic, researching it, writing a script, recording audio (or finding voiceover), sourcing images, editing the video, writing a title and description, finding hashtags, and finally uploading. That's 2–4 hours per video, every single day.

I wanted a way to build a YouTube presence on "educational facts" channels — the kind with millions of subscribers — without the grind. The answer was automation.

The Idea: A 4-Step AI Pipeline

Step 1: Generating the Script with Gemini AI

The script generator calls Google Gemini with a structured prompt, asking it to return JSON with a title, narration script, description, and hashtags. I also built an offline fallback that generates a basic script locally — so the bot never fails completely even during API outages. 

# scripts/generate_script.py (simplified)
import google.generativeai as genai

def generate_script(topic: str):
    model = genai.GenerativeModel("gemini-1.5-flash")
    prompt = f"""Create a YouTube Shorts script about: {topic}
    Return JSON with: title, script (60s), description, tags list"""
    
    response = model.generate_content(prompt)
    return json.loads(response.text)  # Returns clean dict

Step 2: Generating Human-Quality Voiceover

Microsoft Edge TTS is the hidden gem of this whole project. It's completely free, requires no API key, and the neural voices (like en-US-AriaNeural) sound remarkably human. You can control speed, pitch, and choose from dozens of voices including Indian English options. 

# scripts/generate_voice.py (simplified)
import edge_tts, asyncio

VOICE = "en-US-AriaNeural"   # Change to en-IN-NeerjaNeural for Indian accent
RATE  = "+10%"              # Slightly faster — good for Shorts

async def _speak(text, output_path):
    communicate = edge_tts.Communicate(text, VOICE, rate=RATE)
    await communicate.save(output_path)

def generate_voiceover(script: str, output_path: str):
    asyncio.run(_speak(script, output_path))
    return True

Step 3: Assembling the Video with MoviePy

The video generator fetches AI-generated images from Pollinations.ai (no key needed), overlays text slides with channel branding, syncs everything to the audio duration, and outputs a 1080×1920 vertical video ready for Shorts.

Step 4: Uploading to YouTube

The YouTube uploader handles OAuth 2.0 authentication, injects the AI-generated title, description, and tags, sets video category and privacy, and returns the live video URL. The OAuth token is stored locally so you only authorize once — ever.

The Automation Layer: GitHub Actions

This is what makes it truly zero-maintenance. GitHub Actions runs the entire pipeline on a cron schedule — for free — even when your laptop is off, even when you're on vacation, every single day.

📅 GitHub Actions Scheduling Flow
# .github/workflows/auto_upload.yml
name: Daily YouTube Upload

on:
  schedule:
    - cron: "0 10 * * *"   # Every day at 10 AM UTC (3:30 PM IST)
  workflow_dispatch:     # Manual trigger from GitHub UI

jobs:
  upload:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: '3.11'
      - run: pip install -r requirements.txt
      - run: python main.py
        env:
          GEMINI_API_KEY: ${{ secrets.GEMINI_API_KEY }}

How to Set It Up (5-Step Quick Start)

1

Clone the repo and install dependencies

Clone from GitHub, create a virtual environment, run pip install -r requirements.txt. All packages are open source and free.

2

Get your free Gemini API key

Visit aistudio.google.com, sign in with any Google account, and click "Create API Key" in under 60 seconds.

3

Set up YouTube OAuth (one time only)

Create a Google Cloud project, enable YouTube Data API v3, download the OAuth credentials JSON, rename it client_secrets.json, and run the bot once locally to authorize.

4

Customize your topic list

Edit the TOPICS list in main.py. Add your niche — facts, history, science, finance, motivational quotes. The bot cycles through them daily.

5

Push to GitHub and add your secrets

Push the repo, add GEMINI_API_KEY and your encoded YouTube token to GitHub Secrets, and enable Actions. Your bot is live forever.

Make It Your Own: Advanced Customization

The bot is designed to be hacked. Here are the three things most people want to change:

Change the voice

Open scripts/generate_voice.py and change the VOICE variable. Popular choices:

VOICE = "en-US-GuyNeural"     # US male — authoritative, great for facts
VOICE = "en-IN-NeerjaNeural"  # Indian English female — friendly
VOICE = "en-GB-RyanNeural"    # UK male — sounds professional
RATE  = "+15%"              # Slightly faster for Shorts pacing

Change the branding

Edit scripts/generate_video.py:

CHANNEL_NAME = "Your Channel Name"  # Shown in title bar
ACCENT_COLOR = (230, 130, 30)       # Orange — change to match your brand RGB
BG_COLOR     = (245, 242, 235)       # Warm white background

Change upload privacy

Use "private" while testing, switch to "public" for live uploads:

DEFAULT_PRIVACY = "private"   # Safe for testing
DEFAULT_PRIVACY = "public"    # Live — visible on YouTube

What I Learned Building This

Building this taught me three things I didn't expect:

1. The YouTube OAuth setup is the only hard part. Everything else — Gemini, Edge TTS, MoviePy — is laughably easy to integrate. But YouTube's OAuth2 flow is genuinely annoying for first-timers. Stick with it. Once you're past it, the bot runs forever without touching auth again.

2. Edge TTS is wildly underrated. I tried ElevenLabs, gTTS, and even local TTS models. Edge TTS sounds better than most, costs nothing, and has zero rate limits. It shouldn't be free, but it is.

3. Consistency beats perfection on YouTube. My bot's videos aren't as polished as hand-crafted content. But they publish daily, without fail. The algorithm rewards that consistency in ways that occasional "great" videos can't match.
